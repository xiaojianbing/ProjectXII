# ProjectXII 术语表 (Glossary)

> 所有设计文档和代码注释中使用的专业术语必须与本表保持一致。
> 本文件由 `.cursorrules` 引用，请保持同步更新。

---

## 移动与平台跳跃 (Movement & Platforming)

| 术语         | 英文                 | 释义                                                         |
| ------------ | -------------------- | ------------------------------------------------------------ |
| 土狼时间     | Coyote Time          | 角色离开平台边缘后，仍允许跳跃的短暂宽容时间（通常 6-10 帧） |
| 跳跃缓冲     | Jump Buffer          | 角色落地前几帧预按跳跃键，落地瞬间自动起跳                   |
| 边缘修正     | Corner Correction    | 角色头部微微撞到平台边角时，自动水平修正以免被"卡住"         |
| 可变跳跃高度 | Variable Jump Height | 长按跳跃键跳得高，短按跳得低                                 |
| 蹬墙跳       | Wall Jump            | 接触墙壁时按跳跃键，向反方向弹射跳跃                         |
| 边缘攀爬     | Edge Climb           | 手抓住平台边缘后翻身上去的动作                               |

## 战斗与动作 (Combat & Action)

| 术语          | 英文                            | 释义                                                                 |
| ------------- | ------------------------------- | -------------------------------------------------------------------- |
| 打击顿帧      | HitStop / Freeze Frame          | 攻击命中瞬间，双方动画暂停数帧以增强打击感                           |
| 动画取消      | Cancel Window                   | 在特定动画帧数窗口内，允许用另一个动作打断当前动作的后摇             |
| 输入缓存      | Input Buffer                    | 将玩家按键带时间戳地缓存，在当前动作结束后自动执行队列中的下一个指令 |
| 输入队列      | Input Queue                     | 同"输入缓存"，侧重描述多个指令的排队执行                             |
| 受击硬直      | Hit Stun                        | 角色被攻击命中后无法行动的持续时间                                   |
| 击退          | Knockback                       | 角色受击后被推移的距离和方向                                         |
| 挑空          | Launcher                        | 将敌人打飞至空中的攻击类型，用于衔接空中连击                         |
| 空中连击      | Juggling                        | 敌人在空中被连续攻击而无法落地的状态                                 |
| 弹反/完美格挡 | Parry / Perfect Parry           | 在敌人攻击即将命中的精确时间窗口内防御，免伤且可反击                 |
| 势/架势槽     | Posture                         | 用于防守与破防的核心资源，类似《只狼》的"躯干值"                     |
| 破防          | Guard Break                     | 势能耗尽后角色被强制打断动作并进入长硬直                             |
| 穿透伤害      | Chip Damage                     | 即使成功格挡，仍然受到的少量伤害                                     |
| 无敌帧        | I-frames (Invincibility Frames) | 冲刺/闪避过程中角色完全免疫伤害的帧数窗口                            |
| 动作优先级    | Action Priority                 | 定义不同动作之间的打断层级关系                                       |

## 系统与架构 (Systems & Architecture)

| 术语           | 英文                    | 释义                                                                 |
| -------------- | ----------------------- | -------------------------------------------------------------------- |
| 分层有限状态机 | HFSM (Hierarchical FSM) | 将角色状态按层级组织（如 Grounded→Run, Airborne→Fall）的状态管理模式 |
| 运动学刚体     | Kinematic Rigidbody     | Rigidbody2D 设置为 Kinematic 模式，由代码完全控制物理                |
| 射线检测       | Raycast / BoxCast       | 发射虚拟射线检测碰撞，用于替代物理引擎的自动碰撞                     |
| 判定框         | Hitbox / Hurtbox        | 攻击判定区域 (Hitbox) 与受击判定区域 (Hurtbox)，与视觉精灵分离       |
