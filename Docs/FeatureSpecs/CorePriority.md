# ProjectXII 开发优先级与 Feature 规划

## 核心立项理念

**一句话定位**：一款以《忍龙》《鬼泣》级别硬核动作战斗为核心竞争力的 2D Metroidvania。

**开发哲学**：**"先让战斗爽，再让世界活"**
- 核心战斗手感是游戏的**生死线**——如果打击感不够好，再精美的地图和剧情也无法弥补。
- 世界观和叙事为战斗服务——提供情感动机和节奏调节。
- 辅助系统（地图、道具、存档）是基础设施——保障体验流畅，但不是卖点。

---

## 三层优先级金字塔

### 🔴 P0 - 核心支柱：动作战斗手感（必须做到顶级）

这是 **"不做好就不发布"** 的层级。分为 4 个递进子阶段：

#### P0-A：纯动作移动（对标《蔚蓝》）
> **里程碑目标**：在一个灰盒关卡中，仅靠移动就能感到"操控角色本身就很爽"。

| Feature                                      | 竞品参考                                    | 验收标准（玩家视角）                           | 解锁方式   |
| -------------------------------------------- | ------------------------------------------- | ---------------------------------------------- | ---------- |
| 水平移动 + 加速/减速                         | 《蔚蓝》即停即走                            | 零延迟响应，无意外滑行                         | 初始拥有   |
| 下蹲/蹲行/滑铲 (Crouch / Slide)              | 《空洞骑士》下划/《银河战士》滑铲           | 缩小碰撞体穿越低矮通道，滑铲可作为后续攻击前置 | 初始拥有   |
| 跳跃（可变高度 + Coyote Time + Jump Buffer） | 《蔚蓝》《空洞骑士》                        | 短按矮跳、长按高跳，离地后仍可跳               | 初始拥有   |
| 冲刺 (Dash + I-frames)                       | 《蔚蓝》八方向冲刺                          | 冲刺距离精确可控，穿越危险时有"擦弹"爽感       | 初始拥有   |
| 蹬墙跳 + 墙壁滑落                            | 《蔚蓝》Wall Jump                           | 墙面可短暂停留，蹬墙方向精准可控               | 初始拥有   |
| 攀爬 (Edge Grab / Climb)                     | 《蔚蓝》攀爬体力；《Prince of Persia: TLC》 | 抓住边缘后可翻上或松手，有体力限制             | 初始拥有   |
| 二段跳 (Double Jump)                         | 《空洞骑士》君主之翼                        | 空中再次起跳，拓展探索范围                     | 🔒 剧情解锁 |
| 游泳                                         | 《空洞骑士》酸液区域（改为可游泳）          | 水中移动速度降低，有独立的物理手感             | 🔒 剧情解锁 |
| 钩索 (Grapple)                               | 《蝙蝠侠》钩索；《Ori》Bash 弹射            | 锁定钩点后快速拉近，可衔接空中动作             | 🔒 剧情解锁 |

> **开发策略**：所有移动能力的**底层系统和物理手感在 P0-A 统一开发**。标注为"🔒 剧情解锁"的能力，其解锁时机和关卡整合属于 P1 范畴。

#### P0-B：基础战斗（对标《空洞骑士》）
> **里程碑目标**：能用基本攻击砍一个沙包/靶子，每一刀都有令人满足的打击感。

| Feature                               | 竞品参考               | 验收标准                               |
| ------------------------------------- | ---------------------- | -------------------------------------- |
| 普通近战攻击（地面 + 空中）           | 《空洞骑士》骨钉攻击   | 挥空和命中有明显手感差异               |
| 远程射击（基础）                      | 《空洞骑士》复仇之魂   | 远程攻击有独立冷却或资源消耗           |
| 打击反馈（HitStop + 击退 + 屏幕震动） | 《空洞骑士》近战反馈   | 命中瞬间双方顿帧 + 镜头微震 + 击退位移 |
| 受击与无敌帧                          | 《空洞骑士》受击后闪烁 | 被打中后有明确的疼痛反馈和短暂I-frames |
| 基础敌人（沙包/靶子）                 | -                      | 静止的受击对象，用于调试打击感         |

#### P0-C：连招战斗（对标《鬼泣》《忍龙》）
> **里程碑目标**：能打出多段连招且每段有差异化的视觉和手感，高技巧玩家能以华丽的方式击杀敌人。

| Feature                     | 竞品参考                     | 验收标准                            |
| --------------------------- | ---------------------------- | ----------------------------------- |
| 轻/重攻击多段连招           | 《鬼泣5》连招树              | 至少 2-3 条可感知的连招分支路径     |
| 连招派生（轻→重、停顿派生） | 《忍龙Σ》蓄力派生            | 不同输入时机触发不同招式            |
| 挑空 + 空中连击 (Juggling)  | 《鬼泣》空中连击             | 可将敌人浮空并衔接至少 3 次空中攻击 |
| 输入缓存 (Input Buffer)     | 《鬼泣》《忍龙》全系列       | 高速搓招时按键永不丢失              |
| 动画取消 (Cancel Window)    | 《鬼泣》Jump Cancel          | 特定帧窗口内可用跳跃/闪避取消后摇   |
| 连招特效升级                | 《鬼泣5》Style Rank 视觉变化 | 随着连击数增加，特效越来越华丽      |

#### P0-D：保命必杀技（对标《只狼》不死斩）
> **里程碑目标**：玩家在危机时刻释放大招，获得"力挽狂澜"的爽感和安全感。

| Feature            | 竞品参考                              | 验收标准                                       |
| ------------------ | ------------------------------------- | ---------------------------------------------- |
| 大范围必杀技       | 《只狼》不死斩；《鬼泣》Devil Trigger | 巨大视觉特效 + 大范围AOE伤害                   |
| 霸体 (Super Armor) | 《忍龙》绝技蓄力霸体                  | 施放期间不可被打断，受击不硬直                 |
| 能量槽系统         | 《失落的皇冠》能量系统                | 通过击杀、处决、弹反积攒，三种方式积攒速率不同 |
| 演出效果           | 《只狼》不死斩演出                    | 释放瞬间时间变慢/画面变白等戏剧化演出          |

> **已确认的能量槽设计**（参考《Prince of Persia: The Lost Crown》）：
> - **击杀**：少量能量（基础积攒途径）
> - **处决**：中等能量（奖励有意识地处决敌人）
> - **弹反 (Parry)**：大量能量（奖励高技巧操作）
>
> 此机制在 P0-B 阶段需预埋能量槽 UI 和基础数据结构，P0-C 阶段接入处决判定，P0-D 阶段完成消耗释放。
